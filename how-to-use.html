How to Use the System

1. Create a Fruit Batch

The farmer creates a new batch by entering a unique Batch ID, fruit name, and farm location.

2. Update the Batch

Supply chain actors (Processor, Distributor, Retailer) update the batch as it moves through the chain.

3. Track the Batch

Any user can enter a Batch ID to view the full history and verify the fruit’s origin.

4. Verify Blockchain Integrity

This checks whether any record has been tampered with by recalculating block hashes.
